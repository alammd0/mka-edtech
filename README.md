# MKA-EdTech: A Modern E-learning Platform

This is a full-stack e-learning platform built with the MERN stack. It provides a platform for instructors to create and sell courses, and for students to enroll in and consume educational content.

## Tech Stack

*   **Frontend:** React, Redux, Tailwind CSS, Vite
*   **Backend:** Node.js, Express.js
*   **Database:** MongoDB

## Getting Started

### Prerequisites

*   Node.js (v14 or later)
*   npm
*   MongoDB

### Backend Setup

1.  Navigate to the `backend` directory:
    ```bash
    cd backend
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Create a `.env` file in the `backend` directory and add the necessary environment variables (see `.env.example` if available).
4.  Start the backend server:
    ```bash
    npm start
    ```

### Frontend Setup

1.  Navigate to the `frontend` directory:
    ```bash
    cd frontend
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the frontend development server:
    ```bash
    npm run dev
    ```
